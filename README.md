# Mikka Pack

Mikka Pack is a Visual Studio Code extension pack tailored for a Laravel + Tailwind workflow and related productivity tools. Install it once to get your favorite toolkit ready to go.

## Installation

### 1. From the Marketplace

1. Open VS Code → Extensions → search for "Mikka Pack".
2. Click **Install** and let VS Code install the pack along with the bundled extensions.

### 2. From a `.vsix` file

1. Download the `mikka-pack.vsix` file from the releases section.
2. Open VS Code → Extensions → `…` menu → **Install from VSIX...** and select the `.vsix` file.
3. After installation, VS Code will prompt you to install the extensions included in the pack.
